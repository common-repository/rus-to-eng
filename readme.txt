== Description ==

TRANSLATE russian words from post and term slugs to english, or, if Google service is anavaible convert cyrillic in latin. Useful for creating human-readable URLs. 

Work finely with [Cyr-To-Lat](http://wordpress.org/extend/plugins/cyr2lat/) (if Google service is anavaible convert cyrillic in latin).

Based on idea [NeverLex](http://neverlex.com/cat/programming/function-generating-handsome-friendly-url/) and the original plugin Cyr-To-Lat by Atrax, SergeyBiryukov, Anton Skorobogatov. Thanks to Alexander Shilyaev for the idea. (regexp: [Sergey M.](http://iskariot.ru) && [Kama](http://wp-kama.ru), first plugin: [Pensioner](http://1-sites.info))

В отличие от оригинального плагина Cyr-To-Lat, этот плагин не транслитерирует слаги постов и тегов, но переводит их с помощью Google Translate (если тот недоступен - тогда работает транслитерация). Рекомендуется использовать с [Cyr-To-Lat](http://wordpress.org/extend/plugins/cyr2lat/) - в случае недоступности сервиса будет обеспечена транслитерация слагов (а также транслитерация имен файлов), также это обеспечит доступность старых постов.

== Installation ==

1. Upload `rus-to-eng` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
